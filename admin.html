⚙️ 首次配置

管理后台需要连接 GitHub 仓库才能保存内容并自动发布网站。请按以下步骤完成配置（只需做一次）：

1

创建 GitHub 仓库

打开 github.com/new，新建一个仓库（名称随意，选 Public）

2

上传网站文件

在仓库页面点击 uploading an existing file，把 index.html、content.json、admin.html 三个文件拖进去，点击 Commit

3

生成 GitHub Token

打开 github.com/settings/tokens/new，勾选 repo 权限，生成并复制 Token

4

连接 Vercel 到 GitHub

打开 vercel.com → 新建项目 → Import 刚才的 GitHub 仓库，以后每次改完内容会自动发布
